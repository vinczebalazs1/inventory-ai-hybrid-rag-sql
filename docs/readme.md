# Documentation

This folder contains the documentation of the **Inventory AI** project.

Inventory AI is a hybrid AI system that allows users to query university inventory data and technical documentation using natural language.

The system combines multiple AI techniques, including:

- Large Language Models (LLM)
- Text-to-SQL query generation
- Retrieval-Augmented Generation (RAG)
- Vector database search

## Architecture Documentation

The main system architecture and design description can be found in the following document:

- `architecture.pdf` – System architecture and design documentation

- the system architecture
- the query routing mechanism
- the SQL engine
- the RAG engine
- the hybrid query processing workflow
- the technology stack used in the project

## Project Context

This project is developed as part of a **project work at Óbuda University** in the **Artificial Intelligence specialization**.

The goal of the project is to design and implement an AI assistant that can interact with university inventory systems and technical documentation using natural language.
